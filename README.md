Проект посвящён изучению статей о [BatchNorm](https://www.arxiv.org/abs/1502.03167) и [LayerNorm](https://www.arxiv.org/abs/1607.06450), воспроизведению результатов статей и применении при других параметрах.
Работа выполнялась в паре Авдеева Дмитрия и Чернышова Игната вместе с Кузнецовым Владиславым в роли ментора.
